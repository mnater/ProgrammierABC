Aufgabe Schleifen_3 (P **)

Programmiere ein kleines Programm, dass dir mit Hilfe von Buchstaben grafische Muster am Bildschirm ausgibt.
Benutze dazu Schleifen!

Beispiele:

XXXXXX    X       XXXXXXXXXXX          XXX
XXXXX     XX       XXXXXXXXX         XXXXXXX
XXXX      XXX       XXXXXXX         XXXXXXXXX
XXX       XXXX       XXXXX           XXXXXXX
XX        XXXXX       XXX              XXX
X         XXXXXX       X  
















for(i=10; i>0; i--)
{
    s='';
    for(j=i; j>0; j--)
    {
        s=s+'X';
    }
    output(s);
}
