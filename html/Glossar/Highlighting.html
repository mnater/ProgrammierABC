Highlighting
Viele Textprogramme unterstützen den Programmierer bei der Arbeit, indem sie den Programmcode einfärben. Dabei wird der Code selbst nicht geändert, sondern lediglich anders dargestellt.
Beim Einfärben werden z.B. alle Kommentare grün gefärbt. Der Code ist so viel einfacher zu lesen.
Das Highlighting hilft auch bei der Fehlersuche, da Fehler im Code auch oft falsche Farben nach sich ziehen.
