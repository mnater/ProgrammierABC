input(wert);
input() ist eine spezielle Funktion von CodeLab, um Daten im rechten CodeLab-Teil einzugeben.
Diese Funktion funktioniert nur im CodeLab! Wenn du ausserhalb des CodeLabs programmierst, musst du andere Wege finden, um Daten auszugeben.
Beispiele:
output('Bitte einen Namen eingeben:'); name = input(); output('Hallo '+name);
input() darf nicht in einer "while"-, "do while"- oder "for"-Schleife stehen. In diesem Fall ist prompt() zu verwenden.
