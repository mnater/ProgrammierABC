Erste Übung im CodeLab

Es ist höchste Zeit, dass du endlich selbst programmierst!
In diesem Programmierkurs kannst du im CodeLab jederzeit kleine Programme ausprobieren.

Dazu musst du

  1. Das CodeLab öffnen
  2. Auf "Ändern" umschalten
  3. Den Code eintippen
  4. Den Code "Ausführen"

Um einen Wert auszugeben, musst du den Befehl output(wert); verwenden.

Versuch es einmal mit folgendem Code:

pi = 3.14159265358979; text = 'pi = '; output(text); output(pi);
Wenn du im CodeLab auf "Ausführen" oder "Einfärben" klickst, dann wird dein Programmcode eingefärbt.
Das Einfärben nennt man auch Highlighting. Es hilf manchmal bei der Fehlersuche, da nämlich bei ungültigem Code oft auch die Farben falsch sind.
Ändere Werte und Variablennamen zum Beispiel so, dass in einer Variable dein Name gespeichert ist und in einer anderen Variable dein Alter. Gib die Daten so aus, dass zuerst dein Name steht und dann dein Alter.
