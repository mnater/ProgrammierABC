Vordefinierte Funktionen

In jeder Programmiersprache gibt es vordefinierte Funktionen. Einige Funktionen können einfach so verwendet werden, andere stehen erst zur Verfügung, wenn eine bestimmte Funktionsbibliothek in das Programm eingebunden wird.
Die Gesammtheit aller vordefinierter Funktionen nennt man API (Applications Programmer Interface). Je grösser das API ist, desto komplexere Programme kann man schreiben, desto länger geht es aber, bis man die Sprache kennt...

Auch in JavaScript gibt es viele vordefinierte Funktionen. Auf dieser Seite sind die wichtigsten Funktionen aufgelistet und erklärt.

Allgemeine Funktionen

Name Kurzbeschreibung Beispiel
alert() Zeigt ein Informations-Dialogfenster an. alert('Hallo Welt!');
void alert(string Textmedlung)
Die Funktion alert() erwartet einen String-Parameter. Dieser Text wird in einem Dialogfenster ausgegeben.
Das Programm wird angehalten, bis der Benutzer 'OK' drückt.
confirm() Zeigt ein Frage-Dialogfenster an. halt=confirm('Beenden?');
bool confirm(string Textmeldung)
Die Funktion confirm() erwartet als Parameter eine Ja/Nein-Frage. Liefert true oder false zurück.
prompt() Zeigt ein Eingabe-Dialogfenster an. eingabe=prompt('Name:','');
string prompt(string Textmeldung, string Feldvorbelegung)
Die Funktion prompt() zeigt ein Dialogfenster mit einer Eingabeaufforderung, einem Eingabefeld und einem OK- und Abbrechen-Button an.
Erwartet zwei Parameter:
  1. Textmeldung = Text, der angibt, was einzugeben ist.
  2. Feldvorbelegung = Text, der im Eingabefeld steht. Soll das Feld leer sein, muss man eine leere Zeichenkette "" übergeben.

Math-Funktionen

Im Math-Objekt werden verschieden Konstanten und Funktionen für mathematische Berechnungen zusammengefasst. Um auf diese Konstanten oder Funktion zuzugreifen, muss man vor dem Aufruf Math. schreiben:

//Allgemein: //x=Math.Konstante; //x=Math.Funktion(); pi=Math.PI; wurzel3=Math.sqrt(3);
Name Kurzbeschreibung Beispiel
Math.PI Konstante π pi=Math.PI;
number Math.PI
Math.abs() Positiver Wert einer Zahl z=Math.abs(-2)
number Math.abs(number)
Erwartet als Parameter eine Zahl und berechnet den absoluten (positiven) Wert dieser Zahl.
Math.ceil() Rundet eine Zahl auf z=Math.ceil(-3.456)
number Math.ceil(number)
Erwartet als Parameter eine Zahl und rundet diese Zahl auf die nächsthöhere Ganzzahl auf.
Math.floor() Rundet eine Zahl ab z=Math.floor(-3.456)
number Math.floor(number)
Erwartet als Parameter eine Zahl und rundet diese Zahl auf die nächstniedrigere Ganzzahl ab.
Math.pow() Berechnet die Potenz einer Zahl z=Math.pow(2,8)
number Math.pow(number Basis, number Exponent)
Erwartet zwei Zahlen als Parameter. Die erste Zahl ist die Basis, die zweite der Exponent. Gibt Basis hoch Exponent zurück.
Math.random() Zufallszahl zwischen 0 und 1 z=Math.random()
number Math.random()
Gibt eine Bruchzahl zwischen 0 und 1 zurück, z.B. . Erwartet keine Parameter.
Math.round() Rundet eine Zahl auf oder ab. z=Math.round(3.589)
number Math.round(number)
Erwartet eine Zahl als Parameter. Rundet diese Zahl abhängig von der ersten Stelle nach dem Komma auf oder ab.
Math.sqrt() Quadratwurzel z=Math.sqrt(2)
number Math.sqrt(number)
Erwartet eine Zahl als Parameter. Berechnet die Quadratwurzel dieser Zahl.

String-Funktionen

Im String-Objekt werden verschieden Konstanten und Funktionen für die Behandlung von Zeichenketten (Strings) zusammengefasst. Um auf diese Konstanten oder Funktion zuzugreifen, muss man sie mit eine Punkt getrennt hinter eine Zeichenkette schreiben:

//Allgemein: //x=String.Konstante; //x=String.Funktion(); laenge='Hallo Welt'.length; GROSS='Hallo Welt'.toUpperCase();
Name Kurzbeschreibung Beispiel
length Anzahl Zeichen laenge='Hallo Welt'.length;
number String.length
Diese Eigenschaft speichert die Anzahl der Zeichen in einer Zeichenkette. Auch die Leerzeichen zählen!
charAt() Zeichen an Position zeichen='Hallo Welt'.charAt(7);
string String.charAt(number)
Erwartet als Parameter eine Zahl. Liefert das Zeichen der Zeichenkette an der entsprechenden Position.
indexOf() Erste Position einer Zeichenkette zeichen='Hallo Welt'.indexOf('Welt');
number String.indexOf(string)
Erwartet eine Zeichenkette als Parameter. Sucht nach dieser Zeichenkette und gibt an, an welcher Stelle die Zeichenkette als erstes gefunden wurde.
lastIndexOf() Letze Position einer Zeichenkette zeichen='Hallo Welt'.lastIndexOf('Welt');
number String.lastIndexOf(string)
Erwartet eine Zeichenkette als Parameter. Sucht nach dieser Zeichenkette und gibt an, an welcher Stelle die Zeichenkette als letztes gefunden wurde.
slice() Extrahiert Zeichenkette zeichen='Hallo Welt'.slice(0,4);
string String.slice(number Anfang, number Ende)
Erwartet zwei Parameter: den Anfang und das Ende der zu extrahierenden Zeichenkette. Es wird bei 0 zu zählen begonnen. Wenn man den zweiten Parameter weglässt, wird bis zum Ende extrahiert.
substr() Extrahiert Zeichenkette zeichen='Hallo Welt'.substr(0,4);
string String.substr(number Anfang, number Länge)
Erwartet zwei Parameter: den Anfang und die Länge der zu extrahierenden Zeichenkette. Es wird bei 0 zu zählen begonnen. Wenn man den zweiten Parameter weglässt, wird bis zum Ende extrahiert.
toLowerCase() In Kleinbuchstaben umwandeln zeichen='Hallo Welt'.toLowerCase();
string String.toLowerCase()
Wandelt eine Zeichenkette in Kleinbuchstaben um. Erwartet keine Parameter.
toLowerCase() In Grossbuchstaben umwandeln zeichen='Hallo Welt'.toUpperCase();
string String.toUpperCase()
Wandelt eine Zeichenkette in Grossbuchstaben um. Erwartet keine Parameter.
