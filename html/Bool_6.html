AND &&

Der AND-Operator funktioniert wie folgt:

a b a && b
false false false
false true false
true false false
true true true

(a && b) ist nur dann true, wenn a und b true sind.

a=2; b=3; c=4; d = (a<=b); //true e = (b<=c); //true output(d && e); //true
a=2; b=3; c=4; d = (a<=b); e = (b<=c); output(d && e);
