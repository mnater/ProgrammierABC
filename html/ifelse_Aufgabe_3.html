Aufgabe Verzweigungen_3 (P *)

  1. Schreib ein Quiz-Programm, welches mehrere Fragen stellt, die der Benutzer beantworten muss.
    output('Gib Pi auf 4 Stellen genau an'); p = input(); if(p=='3.1415') { output('Richtig!'); } else { output('Falsch!'); }
    Tipp:
    Füge vor dem Programmcode einige Leerzeilen ein, so dass man die Lösungen nicht sieht.
  2. Programmiere Multiple-Choice-Fragen. Der Benutzer soll nur den Buchstaben oder die Nummer der richtigen Antwort eintippen müssen.
  3. Schreib das Programm so, dass am Schluss angegeben wird, wieviel Fragen richtig und wieviel Fragen falsch waren.
